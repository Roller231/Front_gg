## Примечания

1. Все пути в коде начинаются с `/image/` или `/animation/`
2. Файлы в формате `.webp` - оптимизированные изображения
3. Файлы `.json` - Lottie анимации (используются с `@lottiefiles/react-lottie-player`)
4. SVG файлы можно масштабировать без потери качества
